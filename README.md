# Projeto: Conversor de Moedas

## Descrição

Este projeto é uma aplicação simples em Java que realiza a conversão entre moedas com base em uma taxa de câmbio informada pelo usuário.

O programa funciona via terminal, solicitando os dados necessários e exibindo o resultado da conversão de forma organizada.

---

## Funcionalidades

* Entrada de dados pelo usuário via terminal
* Definição da moeda de origem e destino
* Cálculo de conversão com base em uma taxa de câmbio
* Exibição formatada do resultado

---

## Tecnologias Utilizadas

* Java
* Scanner (para entrada de dados)
